Privacy Policy

Effective Date: June 14, 2026 | Last Updated: June 14, 2026

🔒 GrapeDown does not collect, store, transmit, or share any personal data or usage information. Your privacy is fully respected.

Overview

GrapeDown is a desktop download manager application for Windows, accompanied by an optional browser extension for Microsoft Edge and Google Chrome. This privacy policy explains how GrapeDown handles your data across both the desktop application and the browser extension.

Data We Collect

We do not collect any data. Specifically:

  • No personal information (name, email, phone, etc.)
  • No usage analytics or telemetry
  • No download history or browsing activity
  • No crash reports sent to external servers
  • No cookies or tracking identifiers

Network Communications

GrapeDown communicates over the internet solely for the following purposes:

  • Downloading files: The application connects to URLs provided by the user to download files. These connections are made directly to the file hosting servers.
  • Torrent downloads: When downloading via magnet links or .torrent files, the application communicates with BitTorrent DHT routers and peers.
  • Local communication: The browser extension communicates with the desktop application via a local HTTP server on 127.0.0.1 (localhost). This never leaves your computer.

Browser Extension

The GrapeDown browser extension requires the following permissions:

  • downloads: To intercept browser downloads and route them to the GrapeDown desktop application.
  • activeTab / scripting: To detect video elements on web pages and display a download button overlay.
  • host_permissions: Required to intercept download links from any website. No browsing data is collected or transmitted.

The extension only sends download URLs to the local GrapeDown desktop application. No data is sent to any external server.

Data Storage

All data generated by GrapeDown is stored locally on your computer:

  • Downloaded files: Saved to the folder you choose for each download.
  • Temporary chunk files: Created during multi-threaded downloads and automatically deleted upon completion.
  • Error logs: Stored locally in the application directory for debugging purposes. Never transmitted externally.

Children's Privacy

GrapeDown does not knowingly collect any information from children under the age of 13. Since the application collects no personal data, it is inherently compliant with children's privacy regulations.

Third-Party Services

GrapeDown does not integrate with any third-party analytics, advertising, or tracking services.
